Introduction

This is a collection of themes made for Streamlabs Chat Widget. But they are not exactly themes as they are more of an example of features that I'm trying to showcase as i learned how to code them. So each theme usually have different set of features. Code for each features are in the snippets folder. The aim is to help people get more example or documentation on how to design their own style of the widget. You can use the themes as is but you can also mix and match these features manually by editing the code. It's better if you make your own style that match your stream theme.

Getting Started

For Streamlabs web dashboard, not Streamlabs OBS.
  1. Choose your desired theme in the themes folder.
  2. Each theme have 4 main files, HTML, CSS, Javascript (JS) and Custom Fields.
  3. Each files can be opened with notepad (right click > open with > notepad).
  4. Go to your Streamlabs dashboard > All Widgets > Chat Box.
  5. Enable Custom HTML/CSS.
  6. Copy and paste each file to their respective field.
  7. For Custom Field, if it's your first time, you have to click on Add Custom Fields > Edit Custom Fields > paste the code in and then click Update.
  8. You can then edit the options in Custom Fields to modified the theme.
  9. Click Save Settings to save your theme.
  10. Finally, you can copy the widget url and use it as browser source in OBS.
  11. For testing, you can add ?simulate=1 after your widget url to simulate users.
    12. Example: https://streamlabs.com/widgets/chat-box/v1/XXXCODEXXXXX?simulate=1

Streamlabs Settings

  1. Themes Ignore
  2. Badges Choose as you please
  3. Extra Emotes Recommended to enable
  4. Background Color Ignore
  5. Text Color Ignore
  6. Font Size Ignore
  7. Hide Message After Recommended to disable - Some theme might not support this
  8. Chat Delay Choose as you please
  9. Hide Chatter Recommended to disable, or else use theme-name-hide-chatter.js

    10. If you enabled this, use themename-hide-chatter.js file instead of the normal themename.js file because if not, it will ruin the smoothscroll animation. And you must enable both Hide Common Chat Bots & Hide commands starting with `!` option. If there is no themename-hide-chatter.js file in the theme folder, it is not supported yet.

    If you use custom bot names, you might also have to find the code below and add in the bot names.

    const botNames = ["nightbot","moobot","xanbot","deepbot","vivbot","phantombot","streamelements"];

    If you only want to use only one of the option, you have to edit the themename-hide-chatter.js file manually. I did not want include two more files just for it. Look for:

    if (botNames.indexOf(obj.detail.from) == -1 && isCommand != "!"){

    Next, remove one of those according to what you enable, including &&. For example, if you enabled 'Hide Common Chat Bots' but disabled 'Hide commands starting with `!`', your code will look like this:

    if (botNames.indexOf(obj.detail.from) == -1){
  10. Muted Chatters Recommended to disabled*
  11. Custom Bad Words Recommended to disabled*

    12. * If you use both of this option, it might effect the chatbox animation (idk). I never bother with these options so I will not support or troubleshoot this option for you.

Do testing! This overlay are not made for you specificly. I don't know your setup. Please do proper testing to ensure everything ie animation is working. Recommended:
  • Your own message.
  • Commands (!test).
  • Spams quick messages. It is recommended to disable the smoothscroll animation if your chat is fast. IMHO, the animation only looks good in slow chat.
